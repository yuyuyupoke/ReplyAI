description: Create a new feature branch with AI context tracking
---

Create a new feature branch using the auto_feature_branch.sh script.

Usage: /new-branch <type> <name> [description]

Branch types:
- feat: New feature
- fix: Bug fix
- refactor: Code refactoring
- docs: Documentation
- test: Testing
- chore: Maintenance

Example: /new-branch feat ai-improvement "Improve AI reply quality"

Please run: ./scripts/auto_feature_branch.sh with the appropriate parameters.
